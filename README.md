# BunnyEra System

Official website and integration tools for brand ecosystem.

## Features
- Homepage with Text Animator & VideoFrame
- Brand community matrix with SocialShare & SVG icons
- Footer with Framer Social Share Buttons & Smartarget Follow Bar
- Booking module via Bookla
- E-commerce integration with Shopiframe

## Structure
- `src/` → Website source files
- `components/` → Homepage, CommunityMatrix, Shopiframe, Bookla, Footer
- `public/assets/` → Static assets
- `docs/` → Brand guidelines and documentation
